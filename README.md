# Solitaire

## Date: 3/7/22

## By: Daniel Kuntz

---

### **_Description_**

Solitaire is a card game that you play by yourself, hence the name of the game. The goal is to move all the cards into 4 piles at the top of the board.

---

### **_Getting started_**

To start the game, press the "start game" button on.
